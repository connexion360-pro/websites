# Bob Repare

Site vitrine Next.js pour Bob Repare, reparation de trottinettes electriques a Nantes, sur l ile de Nantes.

## Demarrage

npm install
npm run dev

Puis ouvrir http://localhost:3000

## A personnaliser

Remplacer public/logo-placeholder.svg par le logo officiel, ajouter les photos dans public/photos, puis mettre a jour le lien Instagram exact et le telephone quand il sera disponible.
